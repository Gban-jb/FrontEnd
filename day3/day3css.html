üì¶ Box Model BASICS

Four Layers (Outside to Inside)

  1. Margin - Space outside border (transparent)
  2. Border - Line around padding
  3. Padding - Space inside border
  4. Content - Actual content
div {
  margin: 20px;        /* All sides */
  margin: 10px 20px;   /* Top/Bottom Left/Right */
  margin: 10px 20px 30px 40px; /* T R B L (clockwise) */
  
  padding: 15px;
  border: 2px solid black;
}

Box Sizing

* {
  box-sizing: border-box; /* Include padding/border in width */
}
üí° Always use: box-sizing: border-box; on all elements!

üëÅÔ∏è Display Properties BASICS

Common Display Values

  • display: block; - Full width, new line
  • display: inline; - No width/height, same line
  • display: inline-block; - Width/height + same line
  • display: none; - Hidden (removes from layout)
  • display: flex; - Flexbox container
  • display: grid; - Grid container

Visibility

opacity: 0;          /* Invisible but clickable */

üìç Positioning BASICS

Position Values

/* Static (default) - normal flow */
position: static;

/* Relative - relative to normal position */
position: relative;
top: 10px;
left: 20px;

/* Absolute - relative to nearest positioned ancestor */
position: absolute;
top: 0;
right: 0;

/* Fixed - relative to viewport */
position: fixed;
bottom: 20px;
right: 20px;

/* Sticky - switches between relative and fixed */
position: sticky;
top: 0;

Z-Index (Stacking Order)

z-index: 10;  /* Higher = on top */
‚ö†Ô∏è Note: z-index only works on positioned elements!

üé® CSS Grid Basics LAYOUT

üéØ Alignment Cheat Sheet LAYOUT

Flexbox Alignment

  • justify-content: Main axis (horizontal for row)
  • align-items: Cross axis (vertical for row)
  • align-self: Override for single item

Grid Alignment

  • justify-items: Items horizontally in cells
  • align-items: Items vertically in cells
  • justify-content: Entire grid horizontally
  • align-content: Entire grid vertically

Values for All

start | end | center | stretch | space-between | space-around | space-evenly

üêõ Debugging Tips ADVANCED

Browser DevTools

  • Right-click ‚Üí Inspect
  • Look for flex/grid badges
  • Click badge for visual overlay
  • Hover over elements to see box model

Debug CSS

  outline: 1px solid red;
}

  background: rgba(255,0,0,0.1);
}

Common Issues

  • Centering not working? Add height to container
  • Items overflowing? Use min-width: 0
  • Z-index not working? Element must be positioned
  • Margin collapse? Use padding or flex/grid
