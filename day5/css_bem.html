Git Workflow Example

1
Create Feature Branch

Start with a new branch for your feature

git checkout -b feature/css-variables
2
Make Changes

Implement CSS variables and BEM methodology

git add styles.css
3
Commit Changes

Write a descriptive commit message

git commit -m "Add CSS variables and BEM structure"
4
Push to Remote

Upload your feature branch to GitHub

git push origin feature/css-variables
5
Create Pull Request

Open a PR on GitHub for team review

Create PR: feature/css-variables → main
6
Review & Merge

After approval, merge into main branch

git checkout main && git merge feature/css-variables
